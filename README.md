# Frontend Challenge

------------------------------------------------------------------------
# Deploy 
[Vercel](https://frontend-challenge-pi-nine.vercel.app/)
------------------------------------------------------------------------
### Настройка переменных окружения

1. Зарегистрируйтесь и получите ключ на платформе **TheCatAPI**
2. Создайте файл `.env` в корне проекта
3. Добавьте ключ в `.env`:

```bash
VITE_CATS_KEY=your_key
```

# 🚀 Запуск проекта

### 1. Клонировать репозиторий

``` bash
git clone https://github.com/sobselena/frontend-challenge.git
```

### 2. Перейти в папку проекта

``` bash
cd frontend-challenge
```

### 3. Установить зависимости

``` bash
npm install
```
### 5. Запустить проект

``` bash
npm run dev
```

После запуска приложение будет доступно по адресу:

    http://localhost:5173
